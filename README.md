文本分类  
===
【思路】  
--
1、读取数据库中的数据训练模型  
2、使用模型对测试文本进行分类  
3、按分类的标签获取相同标签的文档  
4、计算各文档与测试文档之间的相似程度，并倒序返回  
  
  
【各文件说明】  
---
>conf  
    存储配置信息  
>lib  
    存放一些文档，（如测试数据，停用词表）  
>tools  
    存放一些工具类  
    >>db.py  
        操作数据库工具  
    preprocess.py  
        数据预处理工具（分词，去除停用词，tf-idf选词等操作）  
server.py  
    微服务服务端  
client.py  
    微服务客户端  
demo.py  
    主程序
