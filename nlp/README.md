自然语言处理  
1、读取MySQL中存储的文本数据作为文本库  
2、使用jieba对每篇文本进行分词操作  
3、对文本库中的文本构建词袋模型  
4、使用朴素贝叶斯模型对数据进行训练  
5、预测需要识别文本的类别  
6、使用gensim计算该类别下的所有文本与预测文本之间的相似度，并倒序返回  
  
  
【lib】  
    data_num.txt        存放数据库中数据量，用于比对是否有新数据加入而需要重新训练模型  
    dump.txt            所有数据构建的词袋模型  
    mnb.m               训练好的朴素贝叶斯模型  
    stop_words.txt      停用词列表
